# Getting Started with Cellects

## Installation (Short version)
Install using our Windows installer: [Cellects_installer.exe](https://github.com/Aurele-B/Cellects/releases/)

Or, install via pip:
```bash
pip install cellects
```
Any difficulties? follow our [complete installation tutorial](installation.md)

---

## Quick Start
Run in terminal:
```bash
Cellects
```
